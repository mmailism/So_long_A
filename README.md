-------------------------------- 🖥 How to start game 🖥 --------------------------------
Type 'make' in Terminal.
Type './so_long <name_map_file.ber>'

Can create maps with '0' '1' 'P' 'C' 'E'
'0' is a floor.
'1' is a wall.
'P' is a player.
'C' is a key collect.
'E' is a exit.

Example map:

1111111111111111
100000000C1110C1
1000100100100C11
1E0011C100101111
110C1111C0000P11
1111111111111111

----------------------------------- 👾 How to play 👾 -----------------------------------
press W to move up.
press A to move left.
press S to move down.
press D to move right.
